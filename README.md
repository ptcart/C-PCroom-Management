# 💻 C언어 기반 PC방 관리 프로그램
> **창업프로젝트 1 — 개인 프로젝트**  
> 회원 관리, 시간 관리, 음식 주문, 매출 관리 등  
> PC방 운영에 필요한 기능을 구현한 **C언어 콘솔 기반 프로그램**

---

## 📘 프로그램 개요
본 프로젝트는 C언어를 기반으로 한 **PC방 관리 시스템**으로,  
**사용자**와 **관리자**의 기능을 구분하여 운영 효율성을 높였습니다.  
파일 입출력(File I/O)을 활용해 회원, 주문, 시간 관리 데이터를 저장하며  
큐(Queue) 구조를 통해 음식 주문의 순서를 효율적으로 처리합니다.

---

## 🗂️ 주요 기능

### 👤 사용자 기능
| 기능 | 설명 | 관련 파일 |
|------|------|------------|
| 회원가입 / 로그인 | 사용자 계정 생성 및 인증 | `login.txt` |
| 자리 선택 | 로그인 후 좌석 선택 | `login.txt` |
| 시간 추가 | 결제 후 사용 시간 연장 | `Order.txt`, `login.txt` |
| 먹거리 추가 | 음식 메뉴 선택 및 주문 | `Order.txt`, `Order_detail.txt` |
| 사용 종료 | 사용 종료 후 자동 로그아웃 및 잔여 시간 저장 | `login.txt` |

---

### 🧑‍💼 관리자 기능
| 기능 | 설명 | 관련 파일 |
|------|------|------------|
| 로그인 | 관리자 계정 인증 | `admin.txt` |
| 회원 등록 | 신규 회원 등록 | `login.txt` |
| 회원 수정 / 찾기 | 회원 정보 수정 및 검색 | `login.txt` |
| 회원 삭제 | 회원 정보 삭제 | `login.txt` |
| 회원 시간 추가 | 회원 사용 시간 수정 | `login.txt` |
| 주문 관리 | 음식 주문 내역 확인 및 처리<br>→ **큐(Queue)** 구조 활용 | `Order.txt` |
| 매출 현황 | 매출 합계 출력 및 통계 확인 | - |

## 📁 파일 설계 구조
> 모든 데이터는 **텍스트 파일 기반**으로 관리됩니다.

| 파일명 | 설명 |
|--------|------|
| `login.txt` | 회원 정보 및 로그인 데이터 |
| `admin.txt` | 관리자 계정 정보 |
| `Order.txt` | 음식 주문 내역 |
| `Order_detail.txt` | 주문 상세 내역 |
| `Menu.txt` | 음식 메뉴 목록 |

## 🧠 기술 요소
- **언어:** C  
- **핵심 기술:** 구조체(Struct), 파일 입출력(File I/O), 큐(Queue), 함수 분할(Function Decomposition)  
- **실행 환경:** Visual Studio / Dev-C++  

---

## 🔄 프로그램 흐름 요약
1. **사용자 / 관리자 선택**
2. **로그인 or 회원가입**
3. **사용자: 자리 선택 → 시간 추가 → 음식 주문 → 종료**
4. **관리자: 회원 관리 / 주문 관리 / 매출 확인**
5. **종료 시 자동 저장 (파일 입출력 기반)**

---

## ⚙️ 프로그램 특징
- 파일 기반으로 모든 데이터를 관리 (DB 불필요)
- 큐(Queue)를 활용한 주문 순서 관리
- 실시간 주문 처리 및 자동 종료 기능
- 사용자 편의를 위한 메뉴별 구조적 인터페이스
